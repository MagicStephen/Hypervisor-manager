# Uživatelská dokumentace

Tato část dokumentace slouží jako průvodce používáním aplikace **Hypervisor Manager**.

Popisuje jednotlivé části uživatelského rozhraní a základní operace, které lze v aplikaci provádět.

---

## Co aplikace umožňuje

Aplikace poskytuje jednotné rozhraní pro správu virtualizačních platforem.

Uživatel může:

- spravovat servery a jejich připojení
- pracovat s uzly (nodes) a jejich prostředky
- vytvářet a spravovat virtuální stroje
- používat konzoli pro přímý přístup k VM
- spravovat snapshoty a zálohy
- sledovat logy operací
- využívat konzolové rozhraní (SSH)

---

## Struktura dokumentace

Dokumentace je rozdělena do následujících částí:

- **Servery** – připojení k platformám a práce s infrastrukturou  
- **Uzly** – správa uzlů a jejich prostředků  
- **Virtuální stroje** – správa VM, konfigurace, snapshoty a zálohy  
- **Automatizace** – plánování a správa automatických úloh  
- **Konzole** – práce s CLI rozhraním přes SSH  

---

## Jak začít

1. Přihlaste se do aplikace
2. Přidejte server pomocí tlačítka **+ Add**
3. Připojte se k serveru (Reconnect)
4. Vyberte uzel a začněte pracovat s virtuálními stroji
